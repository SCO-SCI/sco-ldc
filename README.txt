scoldc v3
=========

Quadratic limb-darkening coefficients (u1, u2) for exoplanet transit
photometry, computed by trilinear interpolation of published Claret tables.

Owner: Ed Mullen, Sycamore Canyon Observatory


WHAT IT DOES
------------

Given T_eff, log g, and [Fe/H], scoldc returns quadratic limb-darkening
coefficients for 25 filters (Johnson-Cousins UBVRI JHK, Sloan ugriz,
Stromgren uvby, Kepler, CoRoT, Spitzer S1-S4, TESS, CBB). Interpolation is
trilinear in the three stellar parameters; no extrapolation.

Data sources:

    Tag       Reference                                   Filters          Models
    -------   -----------------------------------------   --------------   ---------------
    CB2011    Claret & Bloemen 2011, A&A 529, A75         23 (see below)   ATLAS + PHOENIX
    C2018     Claret 2018, A&A 618, A20                   TESS             PHOENIX-COND
    CMG2022   Claret, Mullen & Gary 2022, RNAAS 6, 169    CBB              ATLAS

LSM fitting method throughout. Microturbulent velocity xi = 2.0 km/s.

STARTUP CACHE
-------------

Parsing the ~446k-row raw tables takes about 1.3 seconds on a fast CPU and
3-4 seconds on Render's starter tier, which shows up as cold-start latency
after idle. To avoid this, the app loads a pre-parsed pickle
(data/tables.pkl, ~9 MB) at startup instead. A cache hit takes ~150 ms.

The cache is regenerated automatically if it's missing, corrupt, or older
than any of the source .dat files. 

NOTES
-----

- Filter codes 'u' (Stromgren) and 'U' (Johnson) are case-sensitive.
- SDSS filter codes retain their '*' suffix: u*, g*, r*, i*, z*.
- TESS has solar metallicity only; the [Fe/H] input is disabled when TESS
  is selected.
- PHOENIX in CB2011 has solar metallicity only for the filters where it's
  available; [Fe/H] input is disabled in that case too.
- PHOENIX is not available for Spitzer S1-S4 (ATLAS only).
